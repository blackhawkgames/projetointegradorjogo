# PROJETO "?" - PROJETO INTEGRADOR 2026.1

Desenvolvimeto de um jogo educativo com intuito de concientizar pessoas a respeito de decisões no mundo cibernetico de forma lúdica e interativa.

Modelo de "gameplay": history mode com decisões impactando no resultado final da jogatina e ao final, avaliando a perfarmance de "resposabiliade cibernetica" do usuário/jogador

Versão da Unity: 2022.3.13f1
